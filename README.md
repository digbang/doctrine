# Laravel Doctrine

This library aims to ease the implementation of the [Doctrine2 ORM](https://github.com/doctrine/doctrine2)
in a Laravel project.

## Laravel versions

| Laravel version | self.version | self.branch | 
|----------------:|--------------|-------------|
| ~4.2            | ~2.0         | 2.x         |
| ~5.1            | ~3.0         | master      |
